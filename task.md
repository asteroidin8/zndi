> **타겟**: 10~20대 갓생러

> **브랜드**: 극강의 미니멀리즘 · 다크모드 기반 성취감 · Neon Grass Green  
> **최종 갱신**: 2026-06-19

## 앱 구조

### 탭 구조 (4탭)

```
[오늘]    [루틴]    [할일]    [잔디]
```

| 탭 | 역할 |
|---|---|
| **오늘** | 대시보드 — 주간 잔디 + 단식 카드 + 오늘 루틴·할일 (제자리 체크) |
| **루틴** | 루틴 관리 + 체크 (반복 항목) |
| **할일** | 할일 관리 + 체크 (일회성 항목, 그룹 지원) |
| **잔디** | 잔디 달력 + 모든 통계 |

- 설정: ⚙️ 아이콘 → 별도 모달 (탭에 포함하지 않음)
- 단식: 오늘 탭의 카드 → 탭하면 확장 카드로 상세 표시 (화면 전환 없음)

### 기존 대비 변경

| 변경 | 이유 |
|---|---|
| 5탭 → 4탭 | 단식 전용 탭 제거. 단식은 오늘 탭에 통합 |
| PagerView → 탭바 | 좌우 스와이프 탭 전환 제거 → 리스트 스와이프 충돌 해소 |
| 통계 → 잔디 | 탭 이름 변경. 잔디 달력이 최상단, 통계는 하위 |
| 홈에서 제자리 체크 | 루틴·할일을 홈에서 바로 체크 가능 (탭 이동 0) |

```
┌─────────────────────────────────────┐
│  zndi                     🔥3   ⚙️  │  TopBar
├─────────────────────────────────────┤
│  ■ ■ ■ □ · · ·           3/4일    │  주간 잔디 (최상단 고정)
├─────────────────────────────────────┤
│  16h 단식 · 02:34:12         ━65%  │  단식 카드 (컴팩트)
│  [탭하면 카드가 확장되어 상세 표시]   │  idle: "단식 · 16h · 시작 →"
├─────────────────────────────────────┤
│  오늘의 루틴                 2/4  > │  미완료만 표시 (최대 5개)
│  ☐ 영어 단어 30개                  │  제자리 체크 가능
│  ☐ 독서 30분                      │  > 탭 → 루틴 탭 이동
├─────────────────────────────────────┤
│  오늘의 할일                 1/3  > │  미완료만 표시 (최대 4개)
│  🔴 면접 준비                     │  제자리 체크 가능
│  🟡 장보기                        │  > 탭 → 할일 탭 이동
└─────────────────────────────────────┘
```

### 밀도 관리 규칙

- 루틴: 미완료만 기본 표시, 최대 5개. 초과 시 "+N개 더보기"
- 할일: 미완료만 기본 표시, 최대 4개. 초과 시 "+N개 더보기"
- 전부 완료 시: 해당 섹션 "✓ 완료" 한 줄로 축소
- 단식 idle: 한 줄 컴팩트. 진행 중: 2줄 (타이머+바). 탭→확장 카드

### 단식 확장 카드

단식 카드를 탭하면 제자리에서 확장되어 상세 정보 표시. 화면 전환 없음.

확장 시 포함 내용:
- 타이머 (대형)
- 시작/완료 예정 시간
- 프로그레스바
- 칼로리 소모량
- 페이즈 메시지
- 목표 프리셋 (idle 시)
- CTA 버튼 (시작/완료/포기)

## 할일 그룹 기능

### 개요

사용자가 관련된 할일을 하나의 그룹으로 묶어 관리.

```
다이소
├─ 석고보드 앙카 구매
├─ 핸드워시 구매

학교
├─ 과제 제출
├─ 발표 자료 정리
```

### 설계 규칙

| 항목 | 결정 |
|---|---|
| 적용 범위 | 할일 전용 (루틴에는 미적용) |
| 깊이 | 1단계만 (중첩 그룹 없음) |
| 접기/펼치기 | 지원. 기본값은 펼침 |
| 중요도 | 개별 항목의 기존 우선순위(높음/보통/낮음) 유지 + 그룹 추가 |
| 고정(Pin) | 제거 (그룹 도입으로 대체) |
| 네이밍 | "그룹" |

### Drag & Drop

- 그룹 생성/이름 변경/삭제
- 그룹 간 항목 이동
- 그룹 밖으로 꺼내기
- 그룹 내부 순서 변경
- 그룹 자체 순서 변경

### 그룹 진행률

잔디 감성 적용:

```
다이소                    2/4
■ ■ □ □
```

그룹의 완료 비율을 미니 잔디 바로 시각화.

### 정렬 규칙

```
그룹 있는 항목: 그룹 순서 → 그룹 내 우선순위 → 그룹 내 사용자 순서
그룹 없는 항목: 우선순위 → 사용자 순서
```

## 스와이프 UX

### 문제

기존 PagerView의 좌우 스와이프(탭 전환)와 리스트 SwipeActions(완료/삭제)가 충돌.

### 해결

PagerView 제거 → 일반 탭바 네비게이션으로 전환.

- 탭 전환: 탭바 터치 (스와이프 아님)
- 리스트 액션: 좌우 스와이프 유지 (충돌 없음)

## 설계 원칙

### 디자인

- Deep Matte Black (#121212) 기반 다크모드
- Neon Grass Green (#22C55E) 중심 성취 경험
- 극강의 미니멀리즘
- Symbol 중심 UI, 불필요한 텍스트 최소화
- 일관된 디자인 토큰 (spacing, radius, size, opacity)

### UX

- 제자리 행동: 보는 곳에서 바로 체크
- 잔디 퍼스트: 앱을 열면 잔디가 가장 먼저 보임
- 즉시 피드백: 체크 → 잔디 반응 (네온 글로우)
- 부드러운 게이미피케이션: 놓쳐도 벌칙 없음

### 기술

- `colors.ts` + `useThemeColors()` 단일 토큰
- `spacing.ts` 의 radius/size/opacity 토큰 활용
- Zustand + AsyncStorage 상태 관리
- Supabase 인증 + 클라우드 동기화
