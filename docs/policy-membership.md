## 통계 인사이트 (Pro)

### Phase 1: 규칙 기반 인사이트

API 비용 없이 로컬에서 패턴을 감지하여 인사이트를 생성한다.

**제공 주기**: 주간 / 월간

**인사이트 유형**:

| 유형 | 예시 |
|------|------|
| 달성률 변화 | "루틴 완료율이 지난주 대비 12% 올랐어요" |
| 연속 기록 | "운동 루틴 21일 연속 달성 중이에요" |
| 요일별 패턴 | "화요일 달성률이 가장 낮아요 (45%)" |
| 할일 처리 속도 | "이번 주 할일 처리 속도가 빨라졌어요" |
| 단식 패턴 | "평균 단식 시간이 14h → 16h로 늘었어요" |
| 보드 기여도 | "보드에서 가장 꾸준한 멤버예요" |

**구현 방식**:
- 로컬 store 데이터 기반 계산 (추가 서버 비용 없음)
- 통계 화면 상단에 카드 형태로 1~2개 핵심 인사이트 표시
- 주간은 월요일, 월간은 1일에 갱신

### Phase 2: AI 분석 리포트 (추후 검토)

규칙 기반 인사이트의 사용자 반응을 보고 도입 여부를 결정한다.

- **제공 주기**: 월간 1회 또는 연간 리포트
- **비용 구조**: AI API 호출 비용이 사용자당 누적되므로, 빈도를 제한하여 멤버십 가격 대비 역마진 방지
- **내용**: 종합 습관 분석, 개선 제안, 목표 달성 예측 등
- **도입 조건**: Phase 1 인사이트 활성 사용률이 일정 수준 이상일 때

## Pro 구독 해지 시

- 기존 데이터 유지 (삭제 없음)
- Free 제한 초과 콘텐츠 (그룹, 보드 등): 기존 항목 유지, 추가 생성만 제한
- 커스터마이징: 선택했던 테마/모양 유지, 변경 시 Free 옵션으로만 선택 가능
- 보드: 기존 멤버 유지, 인원이 Free 상한 이하로 줄면 이후 Free 상한 적용
- 다시 Pro 구독 시 원래 제한으로 복구
