# Assets (스토어 제출용)

EAS 빌드 전 아래 PNG 파일을 이 폴더에 추가하세요.

| 파일 | 크기 | 용도 |
|------|------|------|
| `icon.png` | 1024×1024 | iOS/Android 앱 아이콘 |
| `splash-icon.png` | 200×200+ | 스플래시 중앙 로고 |
| `android-icon-foreground.png` | 1024×1024 | Android adaptive foreground |
| `android-icon-background.png` | 1024×1024 | Android adaptive background |
| `android-icon-monochrome.png` | 1024×1024 | Android 13+ monochrome |
| `favicon.png` | 48×48+ | Web favicon |

`app.json`의 `splash.backgroundColor`는 라이트 테마 surface(`#ffffff`) 기준입니다.
